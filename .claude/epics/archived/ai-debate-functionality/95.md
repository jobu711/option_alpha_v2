# Task: Add checkbox multi-select to candidates table template

## Description
Add a checkbox column to the candidates table so users can select specific tickers for on-demand debate. Each checkbox carries the ticker symbol as its value. Include a "Select All" toggle in the table header.

## Acceptance Criteria
- [ ] Checkbox column appears as the first column in the candidates table
- [ ] Each checkbox has `name="debate_symbol"` and `value="{{ score.symbol }}"`
- [ ] Header has a "Select All" checkbox that toggles all row checkboxes
- [ ] Clicking a checkbox does NOT trigger the existing `hx-get` row navigation (event propagation stopped)
- [ ] Checkboxes dispatch a custom event (e.g., `debate-selection-changed`) for the dashboard button to listen to

## Technical Details
- **File**: `src/option_alpha/web/templates/_candidates_table.html`
  - Add `<th class="col-select">` with a select-all checkbox in thead
  - Add `<td class="col-select">` with per-row checkbox in tbody (before rank column)
  - Checkbox: `<input type="checkbox" class="debate-check" name="debate_symbol" value="{{ score.symbol }}" onclick="event.stopPropagation()">`
  - Select-all checkbox toggles all `.debate-check` inputs and dispatches event
  - Each checkbox `onchange` dispatches `debate-selection-changed` on document for dashboard.html to react

## Dependencies
- [ ] None (template-only change, works independently)
