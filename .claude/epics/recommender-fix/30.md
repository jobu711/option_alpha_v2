# Task: Relax bid-ask spread default to 30%

## Description
Change the `max_bid_ask_spread_pct` default from `0.10` (10%) to `0.30` (30%) in both the Settings class and the persisted config.json. The current 10% threshold filters out every option contract for non-mega-cap stocks (materials, energy, healthcare sectors have 15-40%+ spreads), causing the options phase to return 0 recommendations.
